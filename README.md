# Yaz Boz Plus

Okey oyunu için puan takip uygulaması.

## Özellikler

- Kullanıcı girişi
- 2 takımlı okey oyunu takibi
- Kalan puan ve ceza girişi
- Otomatik puan hesaplama
- Oyun geçmişi kaydı

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

## Build

```bash
npm run build
```
